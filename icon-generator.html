📋 使用步驟

  1. 把 AClogo-removebg-preview.png 拖入上方框
  2. 調整底色（建議 #111111）與留白比例後確認預覽
  3. 點「一次下載全部 4 張」，得到 4 個 PNG 檔
  4. 把這 4 個 PNG 放進 GitHub repo 根目錄（和 index.html 同層）
  5. 重新部署後安裝 PWA，桌面圖示就會變成你的 logo ✓

ⓘ any = 圖示直接顯示，適合所有平台
ⓘ maskable = Android 自適應圖示，系統會依形狀裁切，logo 縮在安全區內（虛線框內）確保不被切掉
