C

1. 開頭檔與基礎

C語言是一切的根基，許多現代語言（包含C++、C#）都深受它影響。要學C，第一步就是認識「開頭檔」，也就是#include。它的功用是把別人已經寫好的函式庫引入你的程式，最常用的是#include&ltstdio.h>（負責輸入輸出），其他常見的還有&ltstdlib.h>（記憶體管理、亂數等）和&ltmath.h>（數學函式）。

每一支C程式的起點都是int main()，這是主程式進入點，所有邏輯都從這裡開始跑。函式最後要有return 0;，代表程式正常結束，回傳0給作業系統。每一行敘述結束後都要加分號（;），這是C語言的規定，少了分號編譯器就會報錯。

C語言的輸出用printf()，輸入用scanf()，兩者都來自&ltstdio.h>。printf的用法是把要印的內容放在雙引號裡，例如 printf("Hello World\n");，其中\n是換行符號。若要印出變數的值，需要用「格式符」：整數用%d、浮點數用%f、字元用%c、字串用%s，例如 printf("%d\n", a);。

scanf()的用法類似，但要在變數名稱前加上&（取址運算子），例如 scanf("%d", &a);，這是告訴程式要把輸入的值存到變數a的記憶體位置，這個概念在後面講指標時會更深入說明。

本節最後說說「運算子」，和C++幾乎一樣：算術運算子有+、-、*、/、%（取餘數）、++（加1）、--（減1）；關係運算子有>、<、==、>=、<=、!=；邏輯運算子有&&（且）、||（或）、!（否）。這些是所有程式邏輯的基石，要熟練！

2. 變數與資料

「變數」是用來儲存資料的命名空間，C語言是強型別語言，宣告變數時必須指定型態，且變數要在使用前先宣告。基本語法是「型態 名稱 = 初始值;」。

C語言常見的資料型態有：int（整數）、float（單精度浮點數）、double（雙精度浮點數，精確度更高）、char（單一字元，用單引號包住，例：'A'）。C語言沒有原生的string型態，字串是用char陣列來表示，例如 char name[20] = "Alice";。

還有幾個加強版型態：long（更大的整數）、unsigned int（只存正整數，範圍更大）、short（較小的整數，節省記憶體）。用sizeof()可以查看某型態佔用幾個Bytes，例如 sizeof(int) 通常是4。

再來是「陣列」，語法是「型態 名稱[大小];」，例如 int arr[5];。記住索引從0開始，arr[0]是第一個、arr[4]是最後一個。也可以在宣告時直接初始化：int arr[5] = {1, 2, 3, 4, 5};。

C語言字串的操作需要用&ltstring.h>裡的函式，常用的有：strlen()取得字串長度、strcpy()複製字串、strcat()串接字串、strcmp()比較字串。注意C語言的字串結尾有一個看不見的空字元 '\0'，所以char陣列的大小要比字串長度多1。

最後介紹「指標（Pointer）」的入門概念，這是C語言最獨特也最強大的特性。指標是一個儲存記憶體位址的變數，宣告語法是「型態* 名稱;」，用&取得變數的位址，用*取得該位址的值，例如：int a = 10; int* p = &a; // p存的是a的位址，*p的值就是10。

3. 流程控制

「流程控制」決定程式的執行路徑，分為條件判斷和迴圈兩大類。C語言的流程控制語法和C++幾乎完全相同，畢竟C++就是從C演化來的！

條件判斷的主角是if...else if...else，語法是：if（條件）{...} else if（條件）{...} else {... }。條件成立就執行對應的區塊，從上往下依序判斷，找到第一個成立的就跑那塊、其他跳過。

另一個是switch，適合用在「一個變數對應多種固定值」的場景，語法是：switch（變數）{ case 值: ... break; default: ...}。每個case後一定要加break，否則程式會繼續往下執行所有case（這個行為叫fall-through）。

迴圈有三種：for適合已知執行次數的情況，語法是 for（初始; 條件; 更新）{...}；while適合條件未知的情況，語法是 while（條件）{...}，先判斷再執行；do...while則是先執行一次再判斷，語法是 do {...} while（條件）;，保證至少跑一次。

迴圈內可以用break直接跳出整個迴圈，或用continue跳過這一輪剩餘的程式直接進行下一輪。另外，C語言也支援goto語法可以跳到指定的標籤位置，但現代程式設計通常不建議使用，因為會讓程式邏輯變得難以追蹤。

4. 函式

「函式」是C語言模組化程式的核心，把重複使用的程式碼包裝起來取個名字，需要時直接呼叫，不用重複寫。宣告語法是「回傳型態 函式名稱（參數型態 參數名稱）{... return 值;}」。

如果函式不需要回傳值，型態寫void；不需要參數則括號內留空或寫void。C語言的函式必須在使用前先宣告，如果函式定義在main之後，需要在前面先寫「函式原型（prototype）」，例如：int add(int a, int b); // 原型宣告，有分號。

C語言的「參數傳遞」預設是傳值（pass by value），函式收到的是複製品，修改不影響原變數。若要在函式內修改原變數，要用傳指標（pass by pointer），把變數的位址傳進去，例如：void swap(int* a, int* b){int tmp = *a; *a = *b; *b = tmp;}，呼叫時寫 swap(&x, &y);。

陣列傳入函式時比較特別，它自動傳址，函式內修改陣列會影響原本的陣列，所以通常還要額外傳入陣列的長度，因為函式內無法用sizeof取得原陣列大小，例如：void printArr(int arr[], int size){...}。

C語言同樣支援「遞迴」，函式自己呼叫自己。經典範例是費氏數列或河內塔。遞迴的重點是要有終止條件（base case），不然程式會一直呼叫自己直到stack overflow崩潰。遞迴很優雅，但每次呼叫都會佔用一份stack空間，在資源有限的環境要謹慎使用。

5. 進階與記憶體：結構

這章是C語言最有特色的部分，也是讓你真正理解「電腦在做什麼」的關鍵。先深入談「指標（Pointer）」，前面說過指標存的是記憶體位址，它強大的地方在於可以直接操作記憶體，實作高效率的資料結構，也是理解C語言精髓的必修課。

指標可以做「動態記憶體配置」，讓程式在執行時才決定要用多少記憶體，語法用&ltstdlib.h>裡的函式：malloc()配置記憶體（例如配置10個int的空間：int* p = (int*)malloc(10 * sizeof(int));）；用完後一定要用free(p)釋放，不然會造成記憶體洩漏（memory leak）！

再來是「結構（struct）」，它讓你可以把不同型態的資料組合成一個新的型態，有點像C++的class但沒有方法。語法是：struct Student { int id; char name[20]; float score; };，使用時宣告 struct Student s1;，存取成員用.，例如 s1.score = 95.5;。

搭配指標存取結構成員時，可以用箭頭運算子->取代「*+.」，例如：struct Student* sp = &s1; sp->score = 95.5;，這和 (*sp).score 是同樣意思，只是箭頭更簡潔直觀。

最後是「typedef」，可以幫型態取別名，讓程式更好讀，常用在struct上：typedef struct { int id; char name[20]; } Student;，之後就可以直接寫 Student s1; 而不用每次都加struct關鍵字，清爽許多！

恭喜學完C語言的基礎！C語言是所有程式語言的地基，把這裡學好，之後不管學C++、Java還是Python都會更快上手。建議多練習指標和記憶體的操作，這部分是最多人卡關但也最有成就感的地方(加油！)
