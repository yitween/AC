3. 下載 App

本網站已支援 PWA（漸進式網頁應用程式），可安裝到任何裝置的主畫面，離線也能瀏覽學過的頁面。

🤖

Android

直接從瀏覽器安裝 PWA，或使用 PWA Builder 打包為 .apk 安裝檔。

  1. 用 Chrome 開啟本網站
  2. 點右上角 ⋮ 選單
  3. 選「新增至主畫面」
  4. 確認後即完成安裝
📦 用 PWA Builder 產生 APK

進入 PWA Builder → 點「Package for stores」→ 選「Android」→ 下載 .apk。

🍎

iPhone / iPad

iOS 透過 Safari 的「加入主畫面」功能安裝，體驗近似原生 App。

  1. 用 Safari 開啟本網站
  2. 點底部 分享 按鈕 ⬆
  3. 向下捲動，選「加入主畫面」
  4. 點右上角「新增」確認

⚠ iOS 只有 Safari 支援加入主畫面，Chrome 和 Firefox 不適用。

🖥

電腦（Chrome / Edge）

桌機安裝後以獨立視窗開啟，沒有瀏覽器工具列。

  1. 用 Chrome 或 Edge 開啟
  2. 點網址列右側的 安裝 圖示 ⊕
  3. 或從選單選「安裝 AC Coding…」
✈️
