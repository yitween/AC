1. 基礎與運算

Python是目前最熱門的程式語言之一，語法簡潔、學習曲線平緩，在資料科學、人工智慧、網頁後端、自動化腳本等領域都大量使用。Python是「直譯式語言」，不需要編譯，直接執行，也是動態型別語言，宣告變數不用指定型態。

Python的輸出用print()，例如 print("Hello World")。輸入用input()，回傳值是字串，例如 name = input("請輸入姓名：")，若要轉成數字用int()或float()包住。

Python最方便的是「f-string」，在字串前加f，然後用{變數}直接嵌入值，例如：print(f"我叫{name}，今年{age}歲")，超直觀！Python的字串可以用單引號或雙引號，效果相同。

Python的「運算子」和其他語言大致相同，但有幾個特別的：**是次方（例：2**10 = 1024）、//是整數除法（捨去小數）、%是取餘數。邏輯運算子不用符號，直接用英文：and、or、not，更接近自然語言。

Python最大的特色之一是縮排即區塊，不用大括號，程式碼靠縮排（通常4個空格）來判斷哪些程式碼屬於同一區塊。這讓Python程式碼天生就整齊易讀，但縮排錯誤會直接報錯，要養成好習慣！另外，Python不需要每行加分號。

2. 資料結構

Python有幾個超好用的內建「資料結構」，掌握這些，Python的威力才算發揮出來。首先是list（串列），類似其他語言的陣列，但可以混放不同型態，用方括號宣告：fruits = ['apple', 'banana', 'cherry']，索引從0開始，可以用負數索引（-1是最後一個）。

list的常用方法：.append(x)在尾端加一個元素、.insert(i, x)在位置i插入、.remove(x)移除第一個值為x的元素、.pop()移除並回傳最後一個、len()取得長度。Python的「切片（Slice）」也非常強大：fruits[1:3]取索引1到2（不含3），fruits[::-1]整個反轉，超方便！

第二個是dict（字典），以鍵值對（key-value）儲存資料，用大括號宣告：student = {'name': '小明', 'age': 18}，存取用student['name']或更安全的student.get('name')（找不到時回傳None而非報錯），常用方法有.keys()、.values()、.items()。

第三個是tuple（元組），用小括號宣告，和list的差別是不可修改（immutable），例如：point = (3, 5)，適合存放不應該被改動的資料。第四個是set（集合），用大括號但沒有鍵值，自動去除重複，支援數學的聯集|、交集&、差集-運算，非常適合去重和比較兩組資料。

3. 流程控制

Python的「條件判斷」用if...elif...else，注意是elif而不是else if（這是Python特有的縮寫），條件後面加冒號:，例如：if score >= 90: ... elif score >= 80: ... else: ...。

Python也有三元運算子，語法很像英文：值A if 條件 else 值B，例如：result = "通過" if score >= 60 else "不通過"，一行搞定簡單判斷。

迴圈方面，for迴圈是Python最常用的，語法是：for item in 可迭代物件: ...，例如：for fruit in fruits: print(fruit)，直接把list每個元素拿出來用，不用管索引。要跑固定次數用range()：for i in range(10): 跑0到9；range(1, 11): 跑1到10；range(0, 10, 2): 每次跳2。

如果同時要索引和值，用enumerate()：for i, fruit in enumerate(fruits): ...；如果要同時遍歷兩個list，用zip()：for a, b in zip(listA, listB): ...，這些都是Python特有的優雅語法。

while迴圈語法和其他語言類似：while 條件: ...，同樣有break跳出和continue跳過。Python還有個特別的「串列生成式（List Comprehension）」，可以用一行建立list：squares = [x**2 for x in range(10)]，生成0到9的平方數list，非常Pythonic！

4. 函式

Python用def定義函式，語法是：def 函式名稱(參數): ...，例如：def add(a, b): return a + b，呼叫時寫 add(3, 5)。Python函式可以回傳多個值（用逗號分隔，實際上回傳的是tuple），例如：return x, y。

Python支援「預設參數」，在定義時給預設值，呼叫時可以省略：def greet(name, msg="你好"): ...，這樣greet("小明")也不會報錯。還有「關鍵字引數」，呼叫時指定參數名稱：greet(msg="嗨", name="小明")，順序不重要，更清晰。

*args和**kwargs是Python特有的彈性參數語法：*args接收任意數量的位置引數（以tuple接收），**kwargs接收任意數量的關鍵字引數（以dict接收），例如：def func(*args, **kwargs): ...，可以接受任何呼叫方式，寫通用工具函式很好用。

Python有個非常強大的「Lambda函式」，語法是：lambda 參數: 運算式，例如：square = lambda x: x**2，呼叫 square(5) 回傳25，適合需要一個簡單函式但不想正式def的場合，常搭配map()、filter()等高階函式使用。

最後介紹「模組（Module）」，Python的強大之處在於龐大的生態系，用import引入模組：import math 後就能用 math.sqrt()、math.pi 等；from math import sqrt 則直接引入sqrt，不用加前綴。常用標準函式庫：os（檔案系統）、random（亂數）、datetime（時間）、json（JSON處理）。

5. 進階應用

Python支援完整的「物件導向（OOP）」，用class定義類別，語法是：class Dog: ...，建構子固定命名為__init__，第一個參數固定是self（代表物件本身），例如：def __init__(self, name): self.name = name。建立物件：myDog = Dog("旺財")。繼承語法：class 子類別(父類別): ...。

Python有很多「魔術方法（Dunder Methods）」，讓自訂類別支援內建運算子，例如：__str__定義print時的顯示方式、__len__讓len()可用、__add__讓+運算子可用，這讓Python的OOP非常靈活。

Python的「例外處理」用try...except...finally，語法是：try: ... except ValueError as e: print(e) finally: ...，可以針對不同型態的例外分別處理，用raise手動拋出例外。

Python的「檔案讀寫」用open()，推薦用with語法，用完自動關閉：with open('file.txt', 'r', encoding='utf-8') as f: content = f.read()，模式有'r'（讀）、'w'（寫，覆蓋）、'a'（附加）。

最後，Python最強大的是第三方套件生態系，用pip install 套件名安裝，例如：numpy（科學計算）、pandas（資料分析）、matplotlib（繪圖）、requests（HTTP請求）、flask/django（網頁框架）。掌握Python基礎後，選一個你有興趣的方向深入，你會發現Python的世界無邊無際！
