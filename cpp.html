C++

1. 開頭檔、輸入輸出、算術運算

C++是在貝爾實驗室開發，用來讓C語言具物件導向，因此C語言編譯器大都可以編譯C++

在開頭的開頭，要先說明的是「開頭檔」，這是將已建立的函式庫導入環境，最基礎的是#include&ltiostream>；其他像是vector等特殊函式就需要導入，這邊提供一個導入所有函式的函式庫：#include&ltbits/stdc++.h>，可以避免忘記導入要用的函式，缺點是運行時間會變長；最後是int main，這是主程式，你的整個程式架構就是以此為骨架。

再來是「using namespace std;」，它的功用是引入「命名空間」，這樣就不必在每一行程式前加上std::，std是空間名稱，而「命名空間」是要區分不同函式庫中名稱相同的函式，有點像是一個班級，std::是班牌，而using namespace是教室，一群人在教室外需要班牌區分班級，但在班級內就知道這夥人是同一班、哪一班的

接著來聊聊「輸入輸出」，C++中輸入是cin，輸出是cout(有人唸「信」、「騷」，但我個人都唸「西印」、「西澳」)，打法是cin>>、cout<<

本節終段就以「運算子」收尾，運算子可分為「指定運算子」、「算術運算子」、「關係運算子」、「邏輯運算子」(雖然都是拿來算數的就是了ㄎㄎ)

指定運算子目前只需要知道=，可以把一個變數指定(例：a=10;)

而算術運算子包含加減乘除，依序為+、-、*、/，這邊提點神奇的東東「%」(唸mod，不要再那邊啪啪啪了)，它可以用來取餘數、「++」加1、「--」減1；另外，C++是有先乘除後加減的觀念的，所以要先做的部分要用括號包住

關係運算子，顧名思義就是判斷關係，有大於、小於、等於、大於等於、小於等於、不等於，依序為>、<、=、>=、<=、!=

最後是邏輯運算子，有「且」、「或」、「否」，分別為「&&」、「||」、「!」

2. 變數與資料型態、陣列

「變數」就像是一個有名字的盒子，可以用來存放資料，而「資料型態」就是這個盒子能放什麼東西。C++是強型別語言，所以宣告變數時一定要指定型態。

常見的資料型態有以下幾種：int（整數，例：1、-5、100）、float（單精度浮點數，有小數的數字）、double（雙精度浮點數，精確度更高）、char（單一字元，用單引號包住，例：'A'）、string（字串，用雙引號包住，例："hello"）、bool（布林值，只有true或false兩種）。

宣告變數的語法是「型態 名稱 = 初始值;」，例如 int a = 10; 或 string name = "Alice";。變數名稱不能以數字開頭，也不能用保留字（像是int、if等）。

另外還有const關鍵字，可以宣告「常數」，也就是宣告後就不能再改變的值，例如 const int MAX = 100;，這樣程式中就不會不小心改到這個值了。

接下來聊「陣列」，陣列就是把一堆同型態的資料排在一起，用索引（index）來存取，語法是「型態 名稱[大小];」，例如 int arr[5]; 就是宣告一個能放5個整數的陣列。

特別要注意的是，陣列的索引從0開始，所以arr[0]是第一個元素，arr[4]才是第五個，這是初學者最常搞混的地方！存取或修改元素的寫法就是 arr[0] = 10;，非常直觀。

除了基本陣列，C++還有更方便的vector，它是一個動態陣列，大小可以隨時增減，需要導入&ltvector>（或直接用bits/stdc++.h）。宣告方式是 vector<int> v;，要新增元素用 v.push_back(10);，要知道目前有幾個元素用 v.size();，非常好用！

3. 條件判斷、迴圈

「條件判斷」讓程式可以根據不同情況做不同的事，最基本的是if...else，語法是：if（條件）{...} else {... }，如果條件成立就執行第一個區塊，否則執行else的區塊。

當有多個條件要判斷時，可以用else if串接，像是判斷成績等第：90分以上A、80分以上B、70分以上C...這樣一路往下判斷。條件中的判斷式就會用到上一節提到的關係運算子和邏輯運算子，例如 if（score >= 90 && score <= 100）{...}

另一個好用的是switch，適合用在一個變數有很多種固定值要判斷的情況，語法是：switch（變數）{case 值1: ... break; case 值2: ... break; default: ...}，記得每個case後面要加break，不然會一直往下跑！

再來是「迴圈」，讓程式可以重複執行某段程式碼。最常見的是for迴圈，語法是：for（初始值; 條件; 更新）{...}，例如 for（int i = 0; i < 10; i++）就會跑10次，i從0跑到9。

另一個是while迴圈，語法是：while（條件）{...}，只要條件成立就會一直執行，適合「不知道要跑幾次」的情況，但要小心別讓條件永遠成立，不然就會變成無窮迴圈！

還有do...while，和while的差別是它會先執行一次再判斷條件，語法是：do {...} while（條件）;，適合「至少要執行一次」的場景，例如選單顯示。

迴圈中有兩個好用的關鍵字：break可以直接跳出迴圈，continue則是跳過這一輪剩下的程式，直接進行下一次迴圈，善用這兩個可以讓程式邏輯更清晰。

4. 函式

「函式」就是把一段程式碼包裝起來、取個名字，需要的時候就呼叫它，不用每次都重寫一樣的程式，概念很像數學的函數 f(x)，給它一個輸入，它給你一個輸出。

函式的基本語法是：「回傳型態 函式名稱（參數型態 參數名稱, ...）{... return 值;}」，例如寫一個計算兩數之和的函式：int add（int a, int b）{return a + b;}，呼叫時就寫 add(3, 5); 即可。

如果函式不需要回傳值，回傳型態就寫void，此時不需要寫return（或只寫 return; 提早結束函式）。如果不需要任何輸入，括號裡就留空，例如：void sayHello（）{cout << "Hello!";}

函式中的「參數傳遞」有兩種方式：傳值（pass by value）是把資料複製一份傳進去，函式內的修改不影響原本的變數；傳參考（pass by reference）是把原本的變數直接給函式用，在參數名稱前加&，函式內的修改會直接影響原本的變數，例如：void swap（int& a, int& b）{...}

還有一個概念叫「函式多載（Overloading）」，C++允許同名函式有不同的參數型態或數量，編譯器會根據你傳入的參數自動判斷要呼叫哪個版本，有點像同一個人被不同稱呼，但大家都知道是他。

最後，「遞迴（Recursion）」是函式自己呼叫自己的技巧，聽起來很玄，但很多問題用遞迴解起來超優雅，例如計算階乘：int factorial（int n）{if（n <= 1）return 1; return n * factorial（n-1）;}。要注意一定要有終止條件，不然也是無窮迴圈的命運。

5. 類別與物件、繼承、資料儲存、例外處理

終於來到C++最核心的精神——「物件導向（OOP）」。先說「類別（class）」，它就像是一個設計圖，裡面可以定義資料（屬性）和行為（方法）；而「物件（object）」就是根據這個設計圖建造出來的實體，例如設計圖是「狗」，物件就是「你家的柴犬」。

宣告類別的語法是 class Dog {...};，裡面的成員有public（外部可存取）和private（只有類別內部能存取）兩種存取權限，預設是private。特別的是「建構子（Constructor）」，它是和類別同名、沒有回傳值的特殊函式，在建立物件時會自動執行，用來做初始化的工作。

接著是「繼承（Inheritance）」，讓一個子類別可以繼承另一個父類別的屬性和方法，不用重複寫一樣的程式碼，語法是 class 子類別 : public 父類別 {...};。子類別可以覆寫（override）父類別的方法，讓同名函式在不同類別中有不同行為，這就是「多型（Polymorphism）」。

說到「資料儲存」，程式結束後資料就消失了，所以需要寫入檔案來永久保存。C++用fstream函式庫來操作檔案，寫檔用ofstream，讀檔用ifstream，用法和cout/cin很像，例如：ofstream file("data.txt"); file << "Hello"; file.close();。記得用完要close！

最後是「例外處理（Exception Handling）」，程式執行中難免遇到錯誤（像是除以0、開啟不存在的檔案），例外處理讓程式不會直接崩潰，而是優雅地處理錯誤。語法是 try {...} catch（例外型態 e）{...}，把可能出錯的程式放在try裡，出錯時會丟出exception，然後被catch接住並處理，用throw可以手動丟出例外。

以上就是C++基礎的最後一章，恭喜你撐到這裡(撒花)！物件導向是個需要多練習才會內化的概念，建議多動手寫小專案，像是設計一個簡單的學生成績系統，把這章所學的class、繼承、檔案讀寫通通用上，會進步得非常快！
