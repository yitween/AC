JAVASCRIPT

1. 輸出與運算

JavaScript（JS）是唯一能在「瀏覽器」內原生執行的程式語言，幾乎所有網頁的互動效果都是JS做的。搭配Node.js也可以跑在伺服器端，是全端開發的必學語言。JS是動態型別語言，不需要宣告型態。

JS的輸出有幾種方式：console.log()在瀏覽器的開發者工具Console顯示（最常用）；alert()跳出對話框；document.write()直接寫入網頁（實務上很少用）。開啟Chrome開發者工具用F12，Console分頁就能測試JS。

JS的「運算子」和其他語言大致相同，但有個特別重要的：==（寬鬆相等，會自動轉型）和===（嚴格相等，型態也要一樣）。例如："5" == 5 是true，但 "5" === 5 是false。強烈建議永遠用===，==的自動轉型很容易造成難找的bug！

JS也有**（次方）、%（取餘數）、++/--（加減1）。字串可以用+串接，例如："Hello" + " " + "World"；更推薦用「模板字串（Template Literal）」，用反引號（`）包住，${}嵌入變數，例如：`我叫${name}，今年${age}歲`，和Python的f-string概念相同。

2. 變數與物件

JS宣告變數有三種關鍵字：var（舊式，有作用域問題，現在不建議用）、let（現代寫法，可重新賦值）、const（常數，宣告後不可重新賦值）。現代JS建議預設用const，需要改變時才用let，不要用var。

JS的「陣列（Array）」和Python的list類似，混放型態、動態大小，用方括號宣告：const fruits = ['apple', 'banana']。常用方法：.push(x)加到尾端、.pop()移除最後、.shift()移除第一個、.unshift(x)加到最前、.splice()插入或移除、.slice()切片、.length長度。

JS最核心的資料結構是「物件（Object）」，用大括號宣告鍵值對：const student = {name: '小明', age: 18}，存取用student.name或student['name']。物件可以包含函式（這時叫方法），是JS最重要的概念之一，幾乎所有JS程式都在操作物件。

JS有個獨特的概念：「解構賦值（Destructuring）」，可以快速從物件或陣列取出值：const {name, age} = student; 就等於分別宣告name和age；const [first, second] = fruits;。還有「展開運算子（Spread Operator）」...，可以展開陣列或物件，例如合併：const merged = {...obj1, ...obj2};。

3. 流程控制

JS的「條件判斷」和C/C++語法相同：if...else if...else，條件式記得用===而不是==。三元運算子：條件 ? 成立值 : 不成立值，例如：const label = score >= 60 ? "通過" : "不通過";。

JS的switch語法和C相同，一樣要記得加break。特別介紹「短路求值（Short-circuit）」，JS的&&和||除了邏輯判斷，還常用來做條件賦值：const name = user && user.name;（user存在才取name，否則是undefined）；const val = input || "預設值";（input為假值時用預設值）。

迴圈方面，for、while、do...while語法和C相同。遍歷陣列最常用forEach()：fruits.forEach(fruit => console.log(fruit));，或for...of迴圈：for(const fruit of fruits){...}；遍歷物件的鍵用for...in：for(const key in obj){...}。

陣列的函式式操作是現代JS的主流寫法：.map()對每個元素做轉換並回傳新陣列；.filter()篩選符合條件的元素；.reduce()把所有元素累積成一個值；.find()找第一個符合的元素。例如：const doubled = numbers.map(n => n * 2)，超簡潔！

4. 函式

JS的函式有三種寫法：function 宣告（可以在宣告前呼叫，有「提升」效果）、function 表達式（賦值給變數）、「箭頭函式（Arrow Function）」（最現代的寫法，語法是 (參數) => 回傳值）。例如：const add = (a, b) => a + b;，如果函式體超過一行，要加大括號和return。

JS函式是「第一公民（First-class）」，可以把函式當參數傳入另一個函式（叫Callback函式），也可以從函式回傳函式，這是JS函式式編程的基礎。例如：setTimeout(() => console.log("3秒後"), 3000);，這個箭頭函式就是callback。

JS的「作用域（Scope）」規則要注意：let和const有區塊作用域（大括號內有效），var有函式作用域（整個函式內有效）。「閉包（Closure）」是JS的重要概念，內層函式可以存取外層函式的變數，即使外層函式已經執行完畢，這讓JS可以做到很多強大的設計模式。

現代JS（ES6+）的「模組系統」讓程式碼可以分檔案管理：用export匯出函式或變數，用import引入，例如：import {add} from './math.js';。這是現代前端框架（React、Vue）的基礎，學會模組化思考非常重要。

5. 網頁互動與非同步

JS最核心的功能是操作「DOM（Document Object Model）」，把HTML元素轉換成JS物件來操控。常用方法：document.querySelector()（用CSS選擇器選一個元素）、document.querySelectorAll()（選所有符合的元素）、修改內容用.textContent或.innerHTML、修改樣式用.style.屬性。

「事件監聽（Event Listener）」讓你對使用者的操作做反應：element.addEventListener('click', () => {...})，常見事件有click（點擊）、input（輸入）、submit（表單送出）、keydown（按鍵）、mouseover（滑鼠移過）。這是讓網頁「活起來」的關鍵！

JS的「非同步（Async）」是一大特色，因為JS是單執行緒，為了不阻塞畫面，耗時的操作（API請求、計時器）會非同步執行。舊式寫法用Callback，容易形成「Callback Hell」；現代用Promise，或更優雅的async/await語法。

async/await讓非同步程式碼看起來像同步：在函式前加async，耗時操作前加await，例如：const data = await fetch('https://api.example.com')，搭配try/catch處理錯誤，是目前最推薦的非同步寫法。
