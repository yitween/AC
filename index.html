What are some simple and basic programming languages?

C++

C的延伸語言，效能極高，是UE的核心語言，也常用於競賽程式。

Python

最簡易的語言，常用於計算或生成AI。

JavaScript

常用於網頁互動與視覺化計算並輔助HTML。

Java

跨平台物件導向語言，常用於大型企業後端系統與androidApp開發。

HTML

最常見的網頁結構語言。

CSS

負責網頁的樣式與排版，常用於輔助HTML。

C#

微軟開發，語法優雅，是Unity遊戲引擎的主要開發語言。

C

最早的高級語言，效能極高。

linux

部署程式、競賽環境設定，都離不開linux終端機操作。

SQL

無論是寫網頁、寫 APP還是做AI，資料庫都是避不開的。

What can you do with those languages?

Competition

C++、Python常作為APCS、TOI等競賽的語言；競賽導向的學習常偏向對基礎邏輯和解題心法的訓練。

AI

AI最常使用Python編寫，AI導向的學習會偏向大部分觀念和技巧的了解。

Website

HTML、CSS、JS都是製作網站的語言，本網站也是用此三語言製作。

Backend

後端結構大都以Java和linux為主，方便企業與私人做跨平台連結。

System

系統通常使用C語言和SQL，這是因為C語言最早出現且效能極高；而SQL負責資料庫貯存。

Game

C++、C#是目前Unity、UE等遊戲引擎的主流語言，比起程式方塊具有更高的自有度。

Web_crawler_&_Debug

網路爬蟲(web crawler)是幫你自動上網、自動複製貼上資料；Debug(偵錯)是指把程式中語法邏輯錯誤的地方揪出。
