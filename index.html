Learn

What are some simple and basic programming languages?

C++

C 的延伸，效能極高，是 UE 核心語言，也常用於競賽程式。

Python

最簡易的語言，常用於計算或生成 AI。

JavaScript

常用於網頁互動與視覺化計算，輔助 HTML。

Java

跨平台物件導向語言，常用於企業後端與 Android App。

HTML

最常見的網頁結構語言。

CSS

負責網頁的樣式與排版，常輔助 HTML 使用。

C#

微軟開發，語法優雅，是 Unity 遊戲引擎的主要語言。

C

最早的高級語言，效能極高。

SQL

無論寫網頁、App 還是做 AI，資料庫都是避不開的。

Apply

What can you do with those languages?

Competition

C++、Python 常作為 APCS、TOI 等競賽語言，偏重基礎邏輯與解題心法。

AI

AI 最常使用 Python，學習會偏向觀念與技巧的全面了解。

Website

HTML、CSS、JS 是製作網站的三大語言，本站也是如此。

Backend

後端大都以 Java 和 Linux 為主，方便跨平台連結。

System

系統通常使用 C 和 SQL，C 效能極高，SQL 負責資料庫儲存。

Game

C++、C# 是 Unity、UE 等引擎的主流語言，自由度更高。

Crawler & Debug

網路爬蟲自動蒐集資料；Debug 協助找出語法與邏輯錯誤。
